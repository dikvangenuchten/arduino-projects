# JA - 2026 - Closet Lighting
As part of Jeugd Aktief 2026 podium, the control for the lighting.

# Board
- Eletechsup IO22D08
- Arduino


# Goals
- 8 Light groups switchable by buttons
- 8 inputs to be used for:
    - all on
    - all flickering (Various patterns, rotation?, up down)
    - all off
    - Individually turn on/off all lights
